# HHS Press Room Dashboard Icons

This folder contains SVG icons for use in the Power BI dashboard. All icons use HHS primary blue (#005EA2) and are designed to be professional and government-appropriate.

## Navigation Icons

### `icon_home.svg`
**Purpose:** Home/Dashboard page navigation  
**Usage:** Navigation rail, page headers  
**Visual:** Shield/home icon representing the main dashboard

### `icon_releases.svg`
**Purpose:** Press Releases page navigation  
**Usage:** Navigation rail, page headers  
**Visual:** Document icon representing press releases

### `icon_channels.svg`
**Purpose:** Channels/Attribution page navigation  
**Usage:** Navigation rail, page headers  
**Visual:** Network/channels icon with connection points

### `icon_release_detail.svg`
**Purpose:** Release Detail page navigation  
**Usage:** Navigation rail, drillthrough pages  
**Visual:** Detailed document with bullet points

### `icon_dictionary.svg`
**Purpose:** Metric Dictionary page navigation  
**Usage:** Navigation rail, reference pages  
**Visual:** Book/dictionary icon with lines

## Usage in Power BI

1. **As Image Visuals:** Import these SVGs as image visuals in Power BI
2. **As Button Icons:** Use in button visuals for navigation and actions
3. **As Background Elements:** Embed in custom backgrounds
4. **As Tooltips:** Use in tooltip visuals for context

## Color Specifications

- **Primary Color:** #005EA2 (HHS Blue)
- **Background:** Transparent (can be overlaid on any background)
- **Size:** 24x24 viewBox (scalable to any size)

## File Naming Convention

All icons follow the pattern: `icon_[purpose].svg`
